SchedulerFiles

Java utility for copying, moving, comparing, and organizing files.

Quick guide

  1. Select Source using field, Browse, or drag & drop.
  2. Select Dest. Folder is created when needed.
  3. Optional: enter Dest 2. In Copy mode files replicate to both destinations, like software RAID-1.
  4. Choose Copy to preserve source, or disable it for Move.
  5. Choose Original Tree to preserve hierarchy, or Scheduled for year/month/extension.
  6. Press Start. Use Cancel to stop.

Comparison and integrity

  • Compare by content: skips duplicates; uses SHA-256 when verification is enabled.
  • Compare by name: skips same filenames.
  • Verify SHA-256: enabled by default; verifies content before/after transfer.

Dest 2 / software RAID-1

Dest 2 is optional. Copy transfers to Dest, then replicates to Dest 2. Operation fails if either destination fails. Dest 2 must differ from Source and Dest. Move with Dest 2 is not allowed.

If Dest 2 contains extra files, Info asks whether to delete them, copy them into Dest, keep them, or cancel. This is not hardware RAID or an atomic transaction. Check logs and missing files if second disk fails.

Log and safety

_files_scheduler.log is written in each destination with operations, timestamps, sizes, and hashes.

Back up data before Move operations. Software provided as-is, without warranty.
