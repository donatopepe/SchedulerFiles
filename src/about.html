SchedulerFiles

Java utility for copying, moving, comparing, and organizing files.

Features

  • Copy or move files
  • Preserve original hierarchy or organize by year/month/extension
  • Compare by filename, bytes, or SHA-256
  • Verify transferred files with SHA-256
  • Write transaction log in destination directory
  • Command-line and Swing interfaces

Safety

Back up important data before move operations.
